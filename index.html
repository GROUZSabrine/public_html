-------------------------------------------------------
Comment cloner votre projet?
-------------------------------------------------------

-Créez une VM ubuntu server 18.04
-Cloner VM ubuntu server 18.04:  clone lié
-Configurer le clone de VM ubuntu server 18.04:  Réseau -> carte1 : NAT -> carte2 : Réseau privé hôte

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Comment lancer le scripte principale dans ma VM qui permet de déployer votre application (Apache+PHP+MySQL+Application)? -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

-Installez le serveur web Apache:  sudo apt install apache2
-Installez le module PHP:  sudo apt install libapache2-mod-php
-Installez le SGBD MySQL:  sudo apt install mysql-server
-Créez une BD relative à votre projet:  sudo mysql create database db_app;

-> Création d´un utilisateur ayant tous les droits sur cette BD:  create user user_app identified by 'abc'; grant all on db_app.* to user_app@'%';
-> Votre application se connectera à la BD avec les paramètres de cet utilisateur:  mysql -u user_app -p -D db_app -h 192.168.56.20 -P 3306
